# Mahotsav-Website
This repo is for the creation of the website for the upcoming major event of the Photography and Fine Arts sub-council's event, Mahotsav. Here you'll be able to access the raw code and help in building the website together. :)

## Project Structure

- `index.html`: Landing page.
- `pages/`: Contains HTML pages for events, gallery, about, and registration.
- `style.css`: Main stylesheet.
- `script.js`: Main JavaScript file for frontend logic.
- `server-Mahotsav/`: Backend Node.js application.
- `assets/`: Images and other static assets.
